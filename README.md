# Random Password Generator

## Description

A Python-based Random Password Generator that creates secure passwords according to user-defined criteria. Users can choose the password length and select which character types to include, such as uppercase letters, lowercase letters, numbers, and symbols.

## Features

* User-defined password length (minimum 8 characters)
* Include uppercase letters
* Include lowercase letters
* Include numbers
* Include special symbols
* Input validation
* Generate multiple passwords without restarting the program

## Technologies Used

* Python 3
* random module
* string module

## How It Works

1. Enter the desired password length.
2. Choose the character types to include.
3. The program generates a random password based on the selected criteria.
4. Users can generate additional passwords if needed.

## How to Run

1. Install Python 3.
2. Open the project folder in Terminal or Command Prompt.
3. Run:

```bash
python password_generator.py
```
